# Интегральный Проводник (Integral Agent)

Система для диагностики детских травм (0-18 лет) и автоматического подбора практик развития.

## Как запустить:
1. Установите Python.
2. Запустите `python main.py`.
3. Следуйте инструкциям на экране.

## Структура:
- Модуль 1: Опросник (80 вопросов).
- Модуль 2: Координаты развития (Spiral Dynamics/AQAL).
- Модуль 5: Компенсаторные практики.
